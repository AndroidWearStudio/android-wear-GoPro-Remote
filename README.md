GoPro-Remote for Android Wear
=========================

Android Wear Application to control a GoPro with wifi commands (known as [wifi hack](https://github.com/KonradIT/goprowifihack))

This Application was created for the [Android Wear Hackathon](https://plus.google.com/events/ckeil3ivmdq5qokdmj75tbarpvc) by [GDG DÃ¼sseldorf](https://plus.google.com/107373371386267684213/)

## Restrictions

Only tested with GoPro Hero 3
